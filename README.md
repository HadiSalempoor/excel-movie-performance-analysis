# Excel Movie Data Analysis with Pivot Tables

This project analyzes movie industry data using Microsoft Excel. Pivot tables were used to summarize and compare movie performance across genres, directors, actors, release years, and financial metrics, providing insights into trends and key performance indicators.

## Business Objectives

- Analyze movie performance across different genres
- Compare directors and actors based on movie outcomes
- Evaluate revenue, budget, and profitability trends
- Summarize key insights using pivot tables and charts

## Excel Techniques Demonstrated

- Pivot Tables
- Pivot Charts
- Data Aggregation
- Sorting and Filtering
- Financial Analysis
- Comparative Analysis
- Data Visualization

## Files Included

- Excel workbook containing pivot table analyses
- Supporting datasets for movies, genres, directors, and actors
